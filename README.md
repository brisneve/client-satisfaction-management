# Client Satisfaction Measurement (CSM) Dashboard

## Overview

A web-based R Shiny application designed to collect, manage, analyze, and report Client Satisfaction Measurement (CSM) data for the University of the Philippines Cebu. The platform supports ARTA compliance and Citizen’s Charter monitoring through a structured survey workflow, automated analytics, service quality assessment, and centralized administrative reporting.

## Features

### Client Satisfaction Survey
- Multi-step guided survey workflow for improved user experience
- Client profile and transaction information collection
- Citizen’s Charter awareness, visibility, and usefulness assessment
- Service-specific Service Quality Dimension (SQD) evaluation
- Data privacy notice and consent mechanism
- Support for multiple services per transaction
- Assessment of SQD 0–8 indicators
- Automated computation of service quality scores
- Service-level and office-level performance monitoring
- Customer feedback and recommendation collection

### Interactive Dashboard
- Real-time CSM summary statistics
- Demographic analysis by age, sex, region, and client type
- Service Quality Dimension score visualization
- Insights and performance analytics using Plotly
- Office, year, and service-scope filtering capabilities
- Spreadsheet-style coverage monitoring interface
- Service-level tracking of transaction counts and response rates
- Coverage status management and validation
- Documentation of reasons for low survey participation
- Automated sample size and response count calculations

### Administrative Tools
- Secure office-based administrator access
- View, filter, and manage submitted responses
- Review qualitative feedback and remarks
- Generate office-specific reports and summaries
- Download consolidated datasets and reports in Excel format

### Data Management
- CSV-based data storage with automatic initialization
- Structured service and office catalog management
- Year-based reporting and historical records
- Robust handling of survey, demographic, and transaction data

### Technical Highlights
- Built using R Shiny
- Interactive visualizations powered by Plotly
- Data processing with the Tidyverse ecosystem
- Excel report generation using OpenXLSX
- Responsive interface optimized for desktop and mobile devices
